Color Palette

PRIMARY COLOR

#FFFFFF


This is the primary background color for the page, text on the header, and highlighting the user's location in navigation

SECONDARY COLOR

#C1C1C1


This color is used as the background to bring attention to any text element such as a description

ACCENT COLOR

#FF9534


This color is used for buttons, headings with the h3 tag, and accenting active links in navigation

SECONDARY ACCENT COLOR

#929090


This color is used for the main header

TEXT COLOR

#000000


This color is used for the main text content
